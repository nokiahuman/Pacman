# Pacman Game

A classic Pacman game implementation in Python using Pygame.

## Installation

1. Install Python (3.7 or higher)
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Play

1. Run the game:
   ```bash
   python pacman.py
   ```

2. Controls:
   - **Arrow Keys**: Move Pacman (Up, Down, Left, Right)
   - **R**: Restart game (when game over or you win)

## Game Features

- Classic Pacman gameplay
- 4 ghosts with AI that chase Pacman
- Collect pellets to increase score
- Power pellets make ghosts vulnerable
- Lives system (start with 3 lives)
- Win condition: collect all pellets

## Scoring

- Regular pellet: 10 points
- Power pellet: 50 points
- Eating scared ghost: 200 points

Enjoy the game!
